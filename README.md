# Akan Name Generator

### Description
A web-based application that determines a user's **Akan Name** based on their date of birth and gender. In Ghanaian culture, "Akan names" are derived from the day of the week a person is born, carrying deep cultural significance. This tool uses custom JavaScript logic to validate dates and calculate the specific day of the week.

### BDD (Behavior-Driven Development)
| Scenario | Input | Expected Output |
| :--- | :--- | :--- |
| User enters a valid date | 15/08/1995, Male | "Your Akan name is Kwabena" |
| User enters an invalid date | 31/02/1995 | Alert: "Invalid date: Please check the number of days..." |
| User leaves fields empty | [Submit Click] | Form validation triggers or "Invalid date" alert. |
| User selects Female | 15/08/1995, Female | "Your Akan name is Abenaa" |

### Technologies Used
* **HTML5**: Page structure and input forms.
* **CSS3**: Layout styling and UI responsiveness.
* **JavaScript (ES6)**: Date validation logic and day-of-week algorithms.
